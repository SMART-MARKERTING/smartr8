# CLAUDE.md

## Workflow (follow for every task)
1. Read CLAUDE.md first.
2. Read relevant existing files before changing anything.
3. Present a concise plan — what changes, which files, why.
4. Wait for explicit approval before writing any code.
5. After approved changes: run typecheck/build, then commit and push so Cloudflare auto-deploys.
