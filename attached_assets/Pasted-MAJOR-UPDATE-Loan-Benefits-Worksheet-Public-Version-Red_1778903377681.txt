PART D: CALCULATION ENGINE (BOTH VERSIONS)
============================================

1. EXISTING MORTGAGE
   - Required: Balance, Total Monthly Payment
   - Optional: Rate, Escrow, Years Remaining
   - If "no mortgage" was selected in public mode, all values 
     = 0

2. NEW LOAN APR — AUTO-CALCULATED
   Formula:
   - monthlyPayment = PMT(rate/12, term*12, loanAmount)
   - netProceeds = loanAmount - closingCosts
   - APR = solve for rate where 
     PMT(apr/12, term*12, netProceeds) = monthlyPayment
   - Use iterative bisection
   
   Treat ALL closing costs as prepaid finance charges 
   (simplified illustration).

3. EXTRA MONTHLY PRINCIPAL — AUTO-CALCULATED AND LOCKED
   
   For Cash-Out Refi or Rate Reduction:
   - currentTotalOutflow = existingTotalPayment + 
     sum(debt min payments)
   - newTotalOutflow = newLoanPmt + (existingEscrow if known)
   - monthlySavings = currentTotalOutflow - newTotalOutflow
   - extraMonthlyPrincipal = monthlySavings
   
   For Home Equity 2nd Mortgage:
   - currentTotalOutflow = existingTotalPayment + 
     sum(debt min payments)
   - newTotalOutflow = existingTotalPayment + newSecondLoanPmt
   - monthlySavings = currentTotalOutflow - newTotalOutflow
   - extraMonthlyPrincipal applied to SECOND mortgage only
   
   For no-mortgage scenarios:
   - currentTotalOutflow = sum(debt min payments)
   - newTotalOutflow = newLoanPmt
   - monthlySavings = currentTotalOutflow - newTotalOutflow
   
   If monthlySavings is NEGATIVE:
   - Display warning: "With these inputs, the new loan does 
     not produce monthly savings. This program may not be a 
     fit — please adjust the inputs or consult Mykoal 
     directly."
   - Hide the effective rate calculation
   - Keep the "Email My Worksheet" button available (capture 
     the lead anyway, Mykoal can follow up)

4. DEBTS TABLE
   - Columns: Creditor Name, Balance, Minimum Payment
   - NO rate column
   - Auto-sum totals at bottom
