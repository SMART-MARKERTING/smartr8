Build a new Loan Benefits Worksheet tool on smartr8.com. Source 
template is the uploaded HTML file in /templates/ (or wherever 
the user dropped loan_benefits_worksheet.html). Preserve the 
calculation math and visual design EXACTLY. The build adds 
lead capture, server-side PDF generation, email delivery, and 
dual-mode operation (public + private).

============================================
ROUTES TO BUILD
============================================

1. /worksheet — PUBLIC MODE
   - Visitors run the calculator
   - Print and Email buttons require lead capture form first
   - All advisor fields pre-filled with Mykoal's info (hidden 
     by default in a collapsible "Customize" section)

2. /worksheet/internal — PRIVATE MODE (for Mykoal)
   - Same tool, no lead capture wall
   - Print and Email buttons work immediately
   - Advisor fields editable but pre-filled with defaults
   - Add a "Send to Client" form that lets Mykoal enter just a 
     client name + email and the PDF gets emailed directly
   - Protect this route with simple Basic Auth (username + 
     password set via env variables) so it's not crawlable

3. API route /api/worksheet/generate-pdf
   - Server-side endpoint that takes the worksheet inputs as JSON
   - Generates PDF using @react-pdf/renderer OR Cloudflare 
     Browser Rendering API (pick whichever works cleanest on 
     Cloudflare Pages)
   - Returns PDF as base64 or stream

4. API route /api/worksheet/submit-lead
   - Accepts the lead capture form data + worksheet inputs
   - Routes lead to LeadMailbox (same pattern as existing 
     funnels, source: "worksheet")
   - Calls /api/worksheet/generate-pdf to make the PDF
   - Sends the PDF via Resend to the lead's email
   - Returns success/failure to the client

============================================
MOBILE RESPONSIVENESS
============================================

The current HTML template uses fixed 8.5in worksheet width — 
that breaks on mobile.

For SCREEN view: make the worksheet scale responsively. Use 
max-width: 8.5in with overflow-x: auto on smaller screens, OR 
reflow to single-column layout below 768px.

For PDF: keep 8.5in x 11in letter size locked. PDF is always 
the same size regardless of device.

The input panel and lead capture modal should be fully mobile- 
optimized.
